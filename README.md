# Fast Portfolio

Prototipo de portafolio web para desarrollador de software construido con React, TypeScript y Vite.

## Ejecutar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Enfoque visual

- Hero de alto contraste con foco en producto digital
- Layout moderno que mezcla perfil tecnico e impacto de producto
- Seccion de proyectos lista para reemplazar con casos reales
